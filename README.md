# Simple HTML Website

A basic multi-page website built using only HTML. This project is great for beginners learning how to create and link web pages.

## Features

- Home page
- About page
- Services page
- Contact page
- Navigation menu on every page
- Basic HTML contact form
- No CSS or JavaScript

## Project Structure

```
simple-website/
│── index.html
│── about.html
│── services.html
│── contact.html
│── README.md
```

## Getting Started

1. Download or clone the project.
2. Open the `simple-website` folder.
3. Double-click `index.html` or open it in your web browser.
4. Use the navigation links to explore the other pages.

## Technologies Used

- HTML5

## Learning Objectives

This project helps you learn:
- HTML document structure
- Headings and paragraphs
- Links (`<a>`)
- Navigation menus
- Lists
- Forms
- Creating and linking multiple pages
